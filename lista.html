Comandos SQL - Aula

Aprenda os principais comandos SQL de forma clara e objetiva.

Comandos DDL (Data Definition Language)

  • CREATE DATABASE - Sempre use para criar tabelas, e é o primeiro passo para organizar os dados. o comando básico é CREATE DATABASE nome_do_banco_de_dados; CREATE DATABASE meu_projeto; para MySQL. CREATE DATABASE Database_Test; para SQL Server
  • USE - Diz pro sistema em qual banco de dados você quer trabalhar agora. É tipo entrar dentro da pasta que você acabou de criar.
  • CREATE TABLE - usado para criar uma nova estrutura de tabela num banco de dados, definindo o nome da tabela e as colunas com seus respetivos tipos de dados e restrições
  • ALTER TABLE - Serve para mudar a tabela depois que ela já foi criada. Você pode adicionar colunas, mudar nomes, etc.
  • DROP COLUMN - Remove uma coluna de uma tabela.
  • DESCRIBE - Mostra a estrutura da tabela, tipo um resumo do que tem dentro.

Comandos DML (Data Manipulation Language)

  • INSERT INTO - é usado para inserir novos dados em uma tabela. Ou seja, você cria uma linha (registro) com informações dentro de uma tabela. Dentro disso também tem o Volues que é diz o que você quer colocar (os valores correspondentes a cada coluna).
  • UPDATE - serve para alterar dados existentes em uma tabela. Que também usa o SET(define quais colunas serão alteradas e seus novos valores)(usa o where).
  • DELETE FROM - serve para remover dados de uma tabela(usa o where, sem ele apaga todos os dados da tabela). DELETE FROM nome_da_tabela

Cláusulas de Filtro

  • WHERE - O WHERE é usado para filtrar registros em uma tabela. Ou seja, ele permite que você escolha apenas os dados que quer consultar, atualizar ou apagar. Sem WHERE, os comandos podem afetar todos os registros da tabela.
      Operadores comuns com WHERE
    • = → igual
    • > → maior que
    • < → menor que
    • >= → maior ou igual
    • <= → menor ou igual
    • <> ou != → diferente
    • AND → combina duas condições
    • → uma condição ou outra
    • NOT → nega uma condição
    • LIKE → busca por padrão de texto
  • AND, OR, NOT -
      AND
    • Significa “E”.
    • Só traz os dados que cumpram todas as condições ao mesmo tempo.
      • OR
    • Significa “OU”.
    • Traz os dados que cumpram pelo menos uma das condições.
  • NOT - NOT
    • Significa “NÃO”.
    • Traz os dados que não cumprem a condição.
  • LIKE -LIKE
    • O LIKE serve para procurar valores dentro de colunas de texto.
    • Ele funciona junto com símbolos chamados coringas:
    • % → significa qualquer sequência de caracteres
    • _ → significa um único caractere

Consultas e Relacionamentos

  • SELECT - O SELECT serve para consultar dados de uma tabela. É o comando mais usado no SQL, porque é ele que busca e mostra as informações.
  • JOIN - O JOIN serve para juntar dados de duas (ou mais) tabelas relacionadas. Normalmente, usamos quando temos tabelas diferentes que compartilham uma coluna em comum (ex: id).
